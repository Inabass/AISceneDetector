# Project Overview

## プロジェクト概要

AI Scene Detector は、特定シーンだけを含む学習動画から視覚特徴量を抽出し、対象動画内の類似シーンをフレーム単位で推論し、時間区間として統合して切り出す Windows 向け Web UI アプリケーションである。

## 目的

手作業で動画全体を確認して該当シーンを探す作業を削減する。正例動画、負例動画、ユーザーフィードバックを使い、対象シーンの特徴を段階的に改善できる構成にする。

## 想定ユーザー

- 動画素材から特定シーンを繰り返し抽出する個人または制作担当者
- GPU 搭載 Windows PC を利用できるユーザー
- AI モデルの詳細を知らなくても、Web UI から学習、検出、切り出しを行いたいユーザー

## MVP 範囲

- 正例動画と任意の負例動画の登録
- OpenCLIP によるフレーム特徴量生成
- 類似度ベースの初期モデル生成
- 対象動画のフレーム単位推論
- シーン区間生成、サムネイル、プレビュー、切り出し
- ジョブ管理、進捗表示、ログ表示
- モデルバージョン管理とロールバック
- Windows 用 `setup.bat`、`run.bat`

## 対象外

- マルチユーザー権限管理
- クラウド同期
- シーンデータベース機能
- リアルタイム配信解析
- モバイルアプリ

## 完成条件

Windows 上でセットアップ後、Web UI から学習動画を登録し、モデルを作成し、別動画に対して検出を実行し、検出区間をプレビューして動画ファイルとして出力できること。長時間動画でも処理状況、キャンセル、エラー、再開可能性が管理されること。

## 機能要件

- 学習動画を正例、負例として管理する。
- フレーム抽出間隔、リサイズ、バッチサイズを設定できる。
- GPU を使って OpenCLIP 特徴量を生成する。
- Feature Store に特徴量とメタデータを保存する。
- 類似度、クラスタ、分類器を組み合わせて推論する。
- フレームスコアを時間方向に平滑化し、シーン区間を生成する。
- ffmpeg で copy mode または再エンコードによる切り出しを行う。
- ジョブ状態、進捗、ログ、キャンセル、再試行を管理する。
- モデルバージョン、メタデータ、評価結果を保存する。

## 非機能要件

- Windows のパス、文字コード、長いファイル名に配慮する。
- RTX 5090 の GPU メモリを活用しつつ、OOM 時は自動でバッチサイズを下げる。
- SQLite でも破綻しないよう、長時間トランザクションを避ける。
- 例外はユーザー向けメッセージと詳細ログに分ける。
- 大きな動画ファイルをメモリに一括ロードしない。
- UI は長時間処理中も応答性を維持する。
