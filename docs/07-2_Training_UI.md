# Training UI

## 学習画面

Training UI はモデル作成と追加学習の入口である。ユーザーは正例動画、負例動画、モデル名、学習設定を指定して Job を開始する。

## 正例動画アップロード

正例動画は必須である。複数ファイルを登録でき、アップロード後に ffprobe 検証結果、duration、解像度、サムネイルを表示する。

## 負例動画アップロード

負例動画は任意である。未指定でも学習可能だが、誤検出が増える可能性を UI に表示する。負例は後から追加学習できる。

## モデル名

新規モデルではモデル名を必須とする。追加学習では既存モデルと親バージョンを選択する。

## 学習設定

frame interval、batch size、threshold 初期値、cluster count、自動 batch reduction、Feature dtype を設定できる。詳細設定は折りたたみ、既定値で開始できるようにする。

## 学習進捗

進捗はアップロード、検証、フレーム抽出、Feature 生成、モデル生成、評価、保存の段階で表示する。キャンセルは Job に cancel request を送る。

## ログ表示

ジョブログをリアルタイムに近い間隔で取得し、warning と error を強調表示する。ログはコピー、ダウンロードできる。
